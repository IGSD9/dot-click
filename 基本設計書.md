## この書の見方

開発は **Step 1 → 2 → 3 → 4** の順で進める。

```
Step 0  全体像
  ↓
Step 1  環境構築・DB
  ↓
Step 2  ゲーム本体（点クリック）
  ↓
Step 3  ログイン・スコア・ランキング
  ↓
Step 4  見た目・PWA（後回し可）
```

### 開発方針

| 項目 | 方針 |
|------|------|
| **当面のゴール** | ブラウザ（PC / スマホ）でプレイできること |
| **最終形** | PWA（Step 4） |
| **操作** | PC = マウス左クリック / スマホ = タップ |

## Step 0｜まず全体像

### 0-1. モード一覧

| モード | 内容 |
|--------|------|
| **サバイバル** | 制限時間内にクリック。3回ミスでGO。だんだん加速 |
| **20点スピード** | ランダムな点20個をタップ。かかった秒数を計測 |

ホーム画面でモードを選択してプレイ。

### 0-2. ゲームルール（サバイバル）

| ルール | 内容 |
|--------|------|
| 成功 | 点をクリック → スコア +1 → 次の点がランダム位置に出現 |
| 失敗 | 制限時間内にクリックできない → ミス 1 回（ライフ -1） |
| ライフ | **3 回までセーフ**（3 回ミスでゲームオーバー） |
| スコア | クリック成功回数 |
| 難易度 | 成功するほど **点が消えるまでの時間が短くなる** |
| **操作（PC）** | **マウスカーソルで点を左クリック** |
| **操作（スマホ）** | **指で点をタップ** |

---

### 0-1b. 入力デバイス別の操作（必須）

| デバイス | 操作 | 備考 |
|---------|------|------|
| **PC** | マウスカーソルを点の上に合わせ、**左クリック** | 右クリック・ホイールは無効 |
| **スマホ** | 指で点の上を**タップ** | スクロール・ピンチは無効化 |

点の**外**をクリック / タップしてもミスにはならない（ミスは時間切れのみ）。

---

```
ゲーム開始
  ↓
点がランダム位置に出現（制限時間スタート）
  ↓
  ├─ 点をクリック → スコア+1 → 次の点（より短い制限時間）
  └─ 時間切れ → ミス+1
       ↓
       ├─ ミス < 3 → 新しい点を出す
       └─ ミス = 3 → ゲームオーバー
```

### 0-2b. ゲームルール（20点スピード）

| ルール | 内容 |
|--------|------|
| 目的 | ランダムな点を **20個** タップ |
| 計測 | 右上に **経過秒数** を表示（カウントアップ） |
| 制限時間 | なし（時間制限バーは非表示） |
| 点の出現 | タイマー表示エリアには点が出ない |

### 0-3. 使う技術

| 用途 | 技術 |
|------|------|
| アプリ | Next.js（App Router） |
| DB・認証 | Supabase |
| ORM | Prisma |
| UI | Tailwind CSS |
| 将来 | PWA |

### 0-4. 画面一覧

| 画面 | パス |
|------|------|
| トップ | `/` |
| ゲーム | `/game` |
| ランキング | `/leaderboard` |
| ログイン | `/auth/login` |

### 0-5. データ

| データ | 保存先 |
|--------|--------|
| プレイ中 | ブラウザ（React State） |
| ハイスコア | PostgreSQL `Score.totalScore`（クリック数） |

## Step 2｜ゲーム本体

> **ゴール:** ブラウザで点クリックゲームがプレイできる

### 2-1. やること

| # | 作業 |
|---|------|
| 1 | ゲームエリア（画面内の決められた範囲） |
| 2 | ランダム位置に点を表示 |
| 3 | クリック / タップ判定 |
| 4 | 制限時間・ミス 3 回・ゲームオーバー |
| 5 | スコア・残りライフ表示 |
| 6 | 成功ごとに制限時間を短縮 |

### 2-2. 完成チェック

- [ ] 点がランダム位置に出る
- [ ] **PC:** マウスカーソルで点を左クリックできる
- [ ] **スマホ:** 指で点をタップできる（ページがスクロールしない）
- [ ] 時間切れでミスになる（最大 3 回）
- [ ] スコア（クリック数）が表示される
- [ ] 成功が続くほど点の表示時間が短くなる
- [ ] 点のサイズは常に同じ

→ 詳細: [詳細設計書 Step 2](./詳細設計書.md#step-2ゲーム本体)

## Step 3｜ログイン・スコア・ランキング

> **ゴール:** ハイスコア（クリック数）を保存・ランキング表示

| Action | 内容 |
|--------|------|
| `saveScore` | ゲームオーバー時にクリック数を保存 |
| `getLeaderboard` | クリック数 TOP 50 |

スコア = **1 プレイ中のクリック成功回数**。過去最高を更新したときだけ DB 保存。
