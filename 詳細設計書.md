### 2-3. 制限時間

```typescript
function getTimeLimit(successCount: number): number {
  return Math.max(
    MIN_TIME_MS,
    INITIAL_TIME_MS - successCount * TIME_DECREASE_MS
  );
}
```

| 成功数 | 制限時間 |
|--------|---------|
| 0 | 2000ms |
| 5 | 1825ms |
| 10 | 1650ms |
| 20 | 1300ms |

### 2-4. 状態

```typescript
type GamePhase = 'ready' | 'playing' | 'gameover';

type GameState = {
  phase: GamePhase;
  score: number;       // クリック成功数
  misses: number;      // ミス数（0〜3）
  dot: { x: number; y: number } | null;
  timeLimitMs: number;
};
```

### 2-5. 状態遷移

```
ready ──(START)──→ playing（点出現 + タイマー）
playing ──(クリック成功)──→ score++ → 新しい点 + 短いタイマー
playing ──(時間切れ)──→ misses++
  ├─ misses < 3 → 新しい点
  └─ misses >= 3 → gameover
```

### 2-7. 入力（PC・スマホ）

**方針:** `pointerdown` で PC（マウス）とスマホ（タッチ）の両方に対応する。

| デバイス | ユーザーの操作 | イベント | 条件 |
|---------|--------------|---------|------|
| **PC** | マウスカーソルで点を左クリック | `pointerdown` | `pointerType === 'mouse'` かつ `button === 0` |
| **スマホ** | 指で点をタップ | `pointerdown` | `pointerType === 'touch'` |

**実装要点**

```typescript
function handleDotPointerDown(e: React.PointerEvent) {
  // PC: 左クリックのみ
  if (e.pointerType === "mouse" && e.button !== 0) return;
  e.preventDefault();
  e.currentTarget.setPointerCapture(e.pointerId);
  onHit();
}
```

**CSS（タッチ誤操作防止）**

```css
.game-root {
  touch-action: none;       /* スクロール・ピンチ無効 */
  user-select: none;
  -webkit-user-select: none;
}
.game-dot {
  touch-action: manipulation; /* スマホ: ダブルタップズーム防止 */
  cursor: pointer;            /* PC: マウスカーソルが指マークに */
}
```

**動作確認**

- [ ] PC Chrome: マウス左クリックで点が反応する
- [ ] PC: 右クリックでは反応しない
- [ ] iPhone Safari: 指タップで点が反応、画面がスクロールしない
- [ ] Android Chrome: 同上
